Mac自建Blog指南

发表于 2019-11-06 分类于 Hexo 阅读次数：

//安装请参考https://hexo.io/docs/ , 看英文官网,不要看中文的

//初始化,新建个blog文件夹,进入你的blog目录
npx hexo init

//生成网站,根据主题构建整个博客
npx hexo g

//启动服务器,访问http://localhost:4000即可
npx hexo s

//新增文章,成功后terminals会提示你文章新建在哪,在/blog文件夹/source/_posts下面0
npx hexo new “文章名称”

//按 Ctrl+C 停止本地服务器
//再次启动服务器,访问http://localhost:4000即可
npx hexo s

//本地配置修改
打开 /blog文件夹/_config.yml
#Site
title: Ning
subtitle: Android 自留地
description: ‘’
keywords:
author: Ning
language: zh-CN
timezone: Asia/Shanghai

//设置网站的主题,主题商店https://hexo.io/themes/
//商店里很多主题,这里推荐next主题, 页面搜索“next”即可得到搜索结果
$ cd blog文件夹
$ git clone https://github.com/theme-next/hexo-theme-next themes/next

//修改主题
打开 /blog文件夹/_config.yml
theme: landscape 改为 theme: next

//验证下(每次验证都走下这个流程)
npx hexo clean
npx hexo g
npx hexo s

=============接下来就是next主题的定制化=============
参考文章:https://www.jianshu.com/p/efbeddc5eb19

前提(重要!!):有两个配置文件,
一个是/blog文件夹/_config.yml,我们下面称为站点配置文件
另一个是/blog文件夹/themes/next/_config.yml,我们下面称为主题配置文件

列几点我觉得比较有用的
1.设置Schemes的样式(next主题支持四种风格,个人喜欢Mist风格)

打开主题配置文件,然后修改
#Schemes
#scheme: Muse
scheme: Mist
#scheme: Pisces
#scheme: Gemini

2.配置网站的menu
打开主题配置文件,然后修改
menu:
home: / || home
about: /about/ || user
tags: /tags/ || tags
categories: /categories/ || th
archives: /archives/ || archive
#schedule: /schedule/ || calendar
#sitemap: /sitemap.xml || sitemap
#commonweal: /404/ || heartbeat
然后新建打开的几个menu , 打开terminal终端 , 执行
npx hexo new page “tags”
npx hexo new page “about”
npx hexo new page “categories”
生成的目录在/blog文件夹/source 下面 , 可以随意修改index.md文件
比如
—
title: 关于
date: 2019-11-05 20:35:22
categories: Testing #about
type: “about”
—
3.设置头像
打开主题配置文件,然后修改(可以设置在线和本地的,本地的文件存在/blog文件夹/themes/next/source/images 文件夹中)
avatar:
#Replace the default image and set the url here.
url: /images/apple-touch-icon-next.png
#If true, the avatar would be dispalyed in circle.
rounded: false
#f true, the avatar would be rotated with the cursor.
rotated: false

4.文章列表显示阅读全文
打开主题配置文件,然后修改
auto_excerpt:
enable: true
length: 150

5.浏览文章时显示浏览进度
打开主题配置文件,然后修改
back2top:
enable: true
#Back to top in sidebar.
sidebar: false
#Scroll percent label in b2t button.
scrollpercent: true

9.隐藏网页底部 Hexo 强力驱动
打开主题配置文件,然后修改
powered:
# Hexo link (Powered by Hexo).
enable: false
# Version info of Hexo after Hexo link (vX.X.X).
version: false

theme:
# Theme & scheme info link (Theme - NexT.scheme).
enable: false
# Version info of NexT after scheme info (vX.X.X).
version: false

10.页脚添加访客人数和总访问量
打开主题配置文件,然后修改
busuanzi_count:
enable: true
total_visitors: true
total_visitors_icon: user
total_views: true
total_views_icon: eye
post_views: true
post_views_icon: eye

11.关于图片的问题
1、首先打开配置文件，并搜索 post_asset_folder 设置为：

post_asset_folder: true #资源文件夹开启
2、在博客目录下执行
npm install https://github.com/CodeFalling/hexo-asset-image --save
hexo new “一个新的博客” 的时候，会自动建立一个与文章同名的文件夹，可以把与该文章相关的所有资源都放到那个文件夹，方便的资源管理。
3、完成安装后用hexo新建文章命令建立文章，发现_posts目录下面会多出一个和文章名字一样的文件夹。图片就可以放在文件夹下面。结构如下：

_posts
 ├── demo文章
 |   ├── demo1.jpg
 |   ├── demo2.jpg
 |   └── demo3.jpg
 └── demo文章.md
4、只要使用 ![demo文章](demo3.jpg) 插入图片。其中[]里面不写文字则没有图片标题。同时，生成的 html 结构为：

<img src="/2017/12/16/demo文章/demo3.jpg" alt="demo文章">
如需在主页或者列表页、存档页显示图片，详看上面官方资源文件说明和参考链接。

=============推送到远程=============
//发布到远程,修改站点配置文件_config.yml
deploy:
type: git
repo: https://github.com/ningnim/ningnim.github.io.git
branch: master

//执行如下指令(修改配置后推送到远程服务器也是这套指令)
npx hexo clean
npx hexo g
npx hexo d
